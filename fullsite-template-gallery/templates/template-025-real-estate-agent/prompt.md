# AI 網頁生成大師提示詞 (Master Prompt)

這是一個極致調校的 AI 網頁生成提示詞。客戶可以將以下 **## 系統指令與生成提示詞** 複製並貼入先進的 LLM（例如 Claude 3.5 Sonnet、GPT-4o 或 Gemini 1.5 Pro），即可精準生成出與本樣板同等極致水準的 12 頁完全響應式（RWD）靜態品牌網站。

---

## 系統指令與生成提示詞

你是一位世界級的前端開發工程師與 UI/UX 設計大師。你的任務是編寫出一個符合現代美學、高轉化率且完全響應式（RWD）的 12 頁靜態品牌網站。所有的網頁內文、選單、表單、按鈕以及元數據（Metadata）都必須使用**繁體中文（zh-Hant-TW）**撰寫。

### 1. 視覺與美學設計系統 (Aesthetic Design System)
在你的 `assets/css/style.css` 中，必須嚴格宣告並採用以下 CSS 變數（Design Tokens）：
```css
:root {
  --brand-primary: #6688AA; /* 品牌主色 */
  --brand-primary-hover: #557799; /* 主色懸停狀態 */
  --brand-secondary: #F0F0F0; /* 次要背景色 */
  --brand-accent: #FF9900; /* 強調/點綴色 */
  --bg-color: #FFFFFF; /* 網頁主背景色 */
  --dark-bg: #F9F9F9; /* 深色區塊背景 */
  --text-main: #333333; /* 主要文字顏色 */
  --text-light: #FFFFFF; /* 深色背景上的亮色文字 */
  --text-muted: #777777; /* 輔助/灰色細節文字 */
  --border-color: #EEEEEE; /* 框線與分割線顏色 */
  
  --font-title: 'Plus Jakarta Sans', 'Noto Sans TC', sans-serif; /* 標題字體 */
  --font-body: 'Manrope', 'Noto Sans TC', sans-serif; /* 內文字體 */
  
  --radius-card: 24px; /* 卡片圓角 */
  --radius-btn: 50px; /* 按鈕圓角 */
  --border-width: 0px; /* 框線粗細 */

  --box-shadow-hover: 0 20px 50px rgba(0,0,0,0.12); /* 懸停陰影樣式 */
}
```

### 2. 版型家族與佈局規範 (Layout Family Rules)
本網站採用 **B. 左右分割 Split Layout** 佈局設計。請嚴格遵守以下版型架構與 CSS 規範：
- 版型樣式：首屏與服務區塊採用左右分割 (50/50 Grid) 佈局，奇偶數行交替排列方向以豐富視覺層次。
- 採用語義化 HTML5 標籤（`<header>`, `<main>`, `<section>`, `<footer>`）。
- 使用 `.container` 限寬 1200px 居中佈局，搭配 Grid 與 Flexbox 系統實現高彈性排版。
- 網頁卡片懸停時應有平滑的向上平移（`transform: translateY(-8px)`）與陰影加深效果。

### 3. 首頁 (index.html) 區塊結構
首頁必須包含一個 **hero-fullscreen** 首頁首屏，並依序組合以下中段 Content Blocks：
- 首屏 Hero (型態: hero-fullscreen)：
  * 主標題：「探索房仲業務的無限可能」
  * 副標題：「我們提供最專業的房仲業務服務，為您帶來白金、城市感的極致體驗。」
  * CTA 按鈕：「線上立即預約」，連結至 `booking.html`。
  - block_about_split
  - block_services_list
  - block_portfolio_grid
  - block_testimonials_3col
  - block_cta_gradient

區塊詳細架構：
* **block_about_split**: 展示關於品牌故事的左右分割版塊，一側展示照片，另一側展示關於標題及品牌故事。
* **block_services_list**: 服務列表版塊，橫向排列服務，右側帶有定價與詳情按鈕。
* **block_portfolio_grid**: 等寬的作品展示網格版塊。
* **block_testimonials_3col**: 顧客評價網格，展示 3 欄包含五星評分與顧客評論的卡片。
* **block_cta_gradient**: 漸層背景的行動呼籲版塊，包含 CTA 標題與預約按鈕。

### 4. 12 頁完整頁面架構與品牌文案
你必須編寫並交付以下 12 個網頁檔案。所有頁面都必須共享完全相同的導覽列（頂部固定置頂，包含漢堡選單與行動端側邊抽屜導覽網頁）與頁尾（4欄品牌資訊與社群按鈕）。

- **index.html**：品牌首頁。包含上述 Hero 與 Content Blocks。
- **about.html**：關於我們。展示品牌故事與專業團隊。
  * 品牌故事：「擁有多年的專業經驗，致力於提供高品質的服務，讓每一位客戶都能感受到我們的用心。」
  * 團隊成員：專業顧問 A (資深專員 - 擁有 10 年以上相關領域經驗，致力於提供最佳方案。)、專業顧問 B (執行總監 - 專精於流程優化與品質控管，確保每次服務的完美呈現。)
- **services.html**：服務方案與定價。展示三大核心服務：
  * 服務項目：
  * 專業服務一 (定價 $1,000起，特色：針對您的需求量身打造的高品質方案。)
  * 專業服務二 (定價 $2,000起，特色：提供全方位的專業諮詢與實作協助。)
  * 專業服務三 (定價 $3,000起，特色：高階客製化服務，滿足最嚴苛的要求。)
  * 透明定價卡片：體驗方案 ($3,200/單次)、專業推薦方案 ($8,000/包月，加邊框突出)、尊榮全包方案 ($15,000/季繳)。
- **service-detail.html**：單一服務詳情。展示「專業服務一」的深度說明，並帶有「核心特色說明」區塊。
- **portfolio.html**：精選作品。包含「分類篩選標籤按鈕」與作品網格。
- **reviews.html**：客戶好評見證。三欄卡片式排版，展示真實回饋：
- 王先生 (5星評價：「服務非常專業，完全超乎預期，強烈推薦！」)
- 林小姐 (5星評價：「環境舒適，細節用心，是一次非常棒的體驗。」)
- 張先生 (5星評價：「態度親切且效率高，問題都能迅速得到解決。」)
- **faq.html**：常見問題 FAQ。採用互動手風琴樣式折疊面板：
- 問：「服務包含哪些具體項目？」
  答：「我們的服務涵蓋諮詢、規劃與執行，可依照您的需求調整。」
- 問：「收費方式為何？」
  答：「依照所選方案而定，提供現金、刷卡等多種付款方式。」
- 問：「預約後可以取消嗎？」
  答：「可以，請於預約時間前 24 小時通知我們即可。」
- **booking.html**：線上預約系統。採用三步驟互動引導表單（Step 1: 選擇服務/日期/時段；Step 2: 填寫資料；Step 3: 預約成功明細與 Google 地圖嵌入）。
- **process.html**：服務流程。展示標準流程：第一步：深度需求診斷 -> 第二步：匠人執行施作 -> 第三步：極致成果交付與保固。
- **blog.html**：最新消息。展示精選文章卡片與三欄部落格網格：
* 了解房仲業務的最新趨勢 (2026-05-15 發布)：探討今年的產業發展方向與關鍵變革。
* 如何選擇適合自己的服務方案？ (2026-05-01 發布)：為您解析不同方案的優缺點，幫助您做出明智決定。
* 專家分享：提升體驗的五個小技巧 (2026-04-10 發布)：透過專業角度，教您如何獲得更好的服務品質。
- **blog-detail.html**：部落格文章詳情。展示首篇專欄文章的完整閱讀排版、引言區塊與「發表意見」評論表單。
- **contact.html**：聯絡我們。左側諮詢表單，右側聯絡管道（電話專線：02-1234-5678、信箱：info@real-estate-agent.com、地址：台北市信義區信義路五段 7 號）與 Google 地圖。

### 5. 互動 JavaScript 邏輯
必須在 `<script>` 標籤中實現以下純 JS（Vanilla JS）邏輯：
- **行動端導覽**：點擊選單按鈕給行動端抽屜添加 `.active` 類別，點擊關閉按鈕移除。
- **手風琴 FAQ**：點擊折疊面板標頭，動態切換內容高度（`scrollHeight`）與旋轉展開圖標。
- **作品集篩選**：點擊分類按鈕篩選特定 `data-category` 的作品項目。
- **預約引導表單**：多步驟切換，點擊下一步進行基本驗證並最終在 Step 3 渲染預約明細。
- **導覽列動態選單標記**：依據 `window.location.pathname` 自動為當前頁面的 nav 連結添加 `.active` 與品牌主色。

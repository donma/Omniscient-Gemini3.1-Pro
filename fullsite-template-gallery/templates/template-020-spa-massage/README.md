# SPA Massage - SPA 按摩 網頁樣板

本樣板屬於 **A. 大圖沉浸式 Hero** 版型家族，採用標準 12 頁完全響應式（RWD）設計。

## 專案結構
- `index.html` - 品牌首頁
- `about.html` - 關於我們
- `services.html` - 服務方案項目
- `service-detail.html` - 單一方案細節說明
- `portfolio.html` - 精選作品案例
- `reviews.html` - 顧客滿意好評
- `faq.html` - 常見問題解答
- `booking.html` - 線上預約系統
- `process.html` - 服務交付流程
- `blog.html` - 部落格最新消息
- `blog-detail.html` - 部落格專欄文章內頁
- `contact.html` - 聯絡資訊與地圖
- `assets/css/style.css` - 專屬客製化色彩與字體樣式設計檔

## 設計師調整指引
1. **不要打破版型一致性**：本站 12 頁皆繼承了 `A. 大圖沉浸式 Hero` 家族的排版語言。請勿隨意修改 Hero 或 Grids 結構為傳統樣式。
2. **遵守色彩規範**：全站 CSS 變數定義在 `style.css` 的 `:root` 中。修改變數可全局調整配色。
